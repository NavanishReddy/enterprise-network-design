# Enterprise Network Design using Cisco Packet Tracer

##  Overview

This project demonstrates an enterprise network with VLAN segmentation, inter-VLAN routing, DHCP, and ACL-based security.

---

##  Architecture

* 1 Router
* 1 Multilayer Switch (Cisco 3650)
* 4 Access Switches
* 16 PCs
* 1 Management PC

---

##  Features

* VLAN segmentation (HR, ENG, SALES, IT)
* Inter-VLAN routing
* DHCP configuration
* ACL-based security (department isolation)
* Dedicated Management VLAN
* WAN connectivity
* Disabled unused ports

---

##  VLAN Details

* VLAN 10 – HR – 192.168.10.0/24
* VLAN 20 – ENG – 192.168.20.0/24
* VLAN 30 – SALES – 192.168.30.0/24
* VLAN 40 – IT – 192.168.40.0/24
* VLAN 99 – MGMT – 192.168.99.0/24
